Minecraft Battle Royale

A Spigot plugin for 1.16.5 that lets you play a game mode (very) similar to UHC, however it gives you the liberty of defining a combat arena anywhere and specifying the preparation time for some interesting combinations. There can also be multiple matches going on at one time, this plugin can be installed very easily on any server. This is my very first plugin and Java project.
Get the latest version here: Type '/mcbr' in-game to get a list of all the self-explanatory commands.
