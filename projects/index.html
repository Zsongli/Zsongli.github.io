Hey, welcome to the list of (some of) my projects! Don't get too surprised if there's stuff of all kinds here.

Snowflake BOT

A Discord bot made specifically for The Snowflake Team, out of curiosity. It has some generic Discord bot features, but nothing really special. It's open source, but if you want to avoid getting a seizure, please don't look at the music player's code. You can play rock, paper, scissors with it, so that's pretty cool I guess.
You can check it out if you'd like to: Use the 'help' command to get a list of all the features.

Minecraft Battle Royale

A Spigot plugin for 1.16.5 that lets you play a game mode (very) similar to UHC, however it gives you the liberty of defining a combat arena anywhere and specifying the preparation time for some interesting combinations. There can also be multiple matches going on at one time, and all this can be installed very easily on any server. This is my very first plugin and Java project.
Get the latest version here: Type '/mcbr' in-game to get a list of all the self-explanatory commands.

Karesz WASM

A clone of the educational program 'Karesz a robot', made in C++. Now it's available in browsers with WebAssembly using Emscripten thanks to the engine bringing support.
Try it out: Powered by OLC::PGE
